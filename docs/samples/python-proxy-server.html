Python proxy

This folder contains a simple python proxy server which can be used locally to manually test the proxy connection of the SDK. This is by no means a full proxy and only does the basic intercepting and forwarding of calls. For more complex proxy use cases, it is suggested to use production level proxies such as Squid or nginx.
