Carbon Aware SDK

An SDK to enable the creation of carbon aware applications, that do more when the electricity comes from clean low-carbon sources and less when it does not.

Use in your CICD workflow

Highly Recommended - This provides your team with the possibility to deploy your worloads to Regions with least emissions.

Deploy as a REST end point

Highly Recommended - Best for when you can change the code, and deploy separately. This also allows you to manage the Carbon Aware logic independently of the system using it.

Invoke via command line

Best for use with systems you can not change the code in but can invoke command line.

Write code against the .NET library

Best for when you are using .NET, and you have the ability to change the code, and do not have the ability to deploy the WebApi.
